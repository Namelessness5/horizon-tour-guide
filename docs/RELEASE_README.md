# Horizon Guide —— 安装与启动

车载"电台"式导览:在 Forza Horizon 里开车经过真实取景地时，自动播报当地讲解并在画面底部打字幕。

---

## 1. 系统要求

- **Windows 10 / 11，64 位**
- **Forza Horizon 5 或 6**（本体，任意版本）
- 约 **2.5 GB** 空闲磁盘（软件 ~0.2 GB + 数据 ~1.7 GB）
- 不需要单独安装 .NET —— 软件包已自带运行时。

---

## 2. 你会拿到两个压缩包

| 压缩包 | 内容 | 什么时候更新 |
|---|---|---|
| `HorizonGuide-app-win-x64.zip` | 程序本体（exe + 运行时） | 换软件版本时 |
| `HorizonGuide-data.zip` | 地点数据、内容索引、语音音频 | 补充/修改地点内容时 |

两个分开发，是因为软件小、更新频繁，数据大、不常动。换软件时只需重下前一个包。

---

## 3. 安装

1. **解压软件包** `HorizonGuide-app-win-x64.zip` 到任意目录，比如 `D:\HorizonGuide\`。
   里面有 `HorizonGuide.exe` 和一堆运行时文件。

2. **解压数据包** `HorizonGuide-data.zip`，把里面的 `data\` 和 `content\` 两个文件夹
   **放到 `HorizonGuide.exe` 所在的同一个目录**（和 exe 并列）。

3. 装好后目录应该长这样：

   ```
   D:\HorizonGuide\
   ├─ HorizonGuide.exe        ← 双击这个启动
   ├─ (若干 .dll / 运行时文件)
   ├─ README.md
   ├─ data\
   │  └─ survey-drafts.json
   └─ content\
      ├─ content-index.json
      └─ audio\  ...
   ```

   > 关键：`data\` 和 `content\` 必须跟 `HorizonGuide.exe` 在同一层。
   > 程序启动时会从自己所在目录向上找这两个文件夹，找不到就报错退出。

---

## 4. 让游戏把数据发给软件（一次性设置）

软件靠 Forza 的 **Data Out**（遥测输出）知道你开到了哪。在游戏里打开：

**设置 → HUD 与游戏性 → 数据输出（Data Out）**，往下设置：

| 选项 | 值 |
|---|---|
| Data Out | **开 / ON** |
| Data Out IP 地址 | `127.0.0.1` |
| Data Out IP 端口 | `5300` |

（FH5 和 FH6 都在这个位置。设一次即可，游戏会记住。）

---

## 5. 启动与操作

1. 先启动游戏、进入自由驾驶（确认上面 Data Out 已开）。
2. 双击 **`HorizonGuide.exe`**。屏幕上不会弹主窗口——它是覆盖在游戏画面上的透明字幕层。
3. 用热键控制（全局生效，游戏在前台也能按）：

   | 热键 | 作用 |
   |---|---|
   | **F6** | 开 / 关漫游（开启后才会自动播报） |
   | **F10** | 打开 / 关闭设置窗口（语言、音量、字幕字号等） |
   | **F11** | 跳过当前这段讲解 |

4. 按 **F6** 开启漫游，然后开车经过地图上的取景地，讲解和字幕就会自动出现。

退出：关闭设置窗口只是把它藏起来，程序仍在后台运行。要真正退出，按 **F10** 打开设置窗口，点底部的 **“退出程序”** 按钮。

---

## 6.（可选）用 OBS 录制

- **画面**：用 **"显示器采集 / Display Capture"**。字幕是置顶透明层，会直接盖在游戏画面上，一起录进去。不要用"窗口捕获"抓字幕层（它是隐藏式工具窗，多半抓不到或黑屏）。
- **讲解声音**：加一个 **"应用程序音频采集"**，目标选 **`HorizonGuide.exe`** 进程。
- **游戏声音**：另配一个应用程序音频采集选游戏进程，或用系统音频输出采集。

---

## 7. 常见问题

**双击后闪一下就没了 / 提示找不到数据。**
`data\` 或 `content\` 没和 `HorizonGuide.exe` 放在同一目录。回到第 3 步核对目录结构。

**开了漫游，开车却什么都不播。**
多半是 Data Out 没生效：确认游戏里 Data Out = 开、IP = `127.0.0.1`、端口 = `5300`，且你正在自由驾驶（不是在菜单/回放里）。也确认端口 5300 没被别的遥测工具占用。

**想看运行日志排查问题。**
用命令行带 `--debug` 启动，会弹出一个控制台并把日志写到 `data\roaming.log`：

```
HorizonGuide.exe --debug
```

**字幕位置 / 字号 / 音量想微调。**
按 **F10** 打开设置窗口调整；也可用命令行参数（如 `--font=30 --bottom=140 --volume=80`）启动。
